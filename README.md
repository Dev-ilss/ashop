### Installation
1. Instalar una version de python 3+.
1. Crear un directorio donde guardar el proyecto.
1. Clonar el codigo del repositorio
`git clone https://github.com/Dev-ilss/ashop.git`
1. Entrar a la carpeta de ashop
1. Instalar las dependencias
`pip install -r requirements.txt`
1. Migrar la base de datos
`python manage.py migrate`
`python manage.py makemigrations`
1. Iniciar el servidor 
`python manage.py runserver`
1. Entrar a la pagina en la direccion 127.0.0.1:8000
